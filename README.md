# Aquatic Life Detection using Machine Learning

## Overview
This project uses a CNN-based transfer learning model (MobileNetV2) to classify aquatic animals from images.

## Features
- Multi-class classification (8+ species)
- Transfer learning
- Batch image prediction
- Confidence scoring

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Google Colab

## How to Run
1. Load model
2. Run prediction on images

## Future Work
